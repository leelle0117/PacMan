# Neon Pac-Man

A classic Pac-Man game with a modern neon aesthetic.

## Play Now

Check out the live game here: [https://leelle0117.github.io/PacMan/](https://leelle0117.github.io/PacMan/)

## Features

- **Neon Aesthetic**: Modern visual style with glow effects.
- **D-pad Support**: Optimized for mobile and touch devices.
- **Pause/Resume**: Toggle the game status at any time.
- **Tunnel Wraparound**: Classic gameplay mechanics including side-tunnel movement.

## Controls

- **Desktop**: Use **Arrow Keys** to move.
- **Mobile**: Use the **On-screen D-pad**.
- **Pause**: Click the **⏸** button in the center of the D-pad.

## Local Development

1. Clone the repository.
2. Open `index.html` in a browser or serve using a local server:
   ```bash
   npx serve .
   ```
